Welcome to ImageSort

This is an application designed to help sort images quickly into categories using a drag and drop interface. It is primarily designed for use with a vertical iPad sized screen, so it may not scale as well on other screen sizes.

Creating Groups

To create a new group, tap the black plus in the upper right of the groups interface. Up to five groups can be added. Children can be added by taping the "+" in the top right.

Editing a group

The groups currently load with pictures in them. This is because there is a bug with the drag and drop when the container is empty. To remove images or change the group name, tap and hold the name of the group to bring up the group edit window. DO NOT remove all the images. You will not be able to drag any new images into this group. Remove individual images by tapping. This also shows all images that are in the group (the main page only shows the most recent images that have been added.

Deleting a group

Groups can only be deleted when all of their children are deleted. Close all sub groups (by using the "x" in the top right) then close the parent group.

Sorting images

Images can be dragged from the column on the right into any of the groups. Images on the right can also be reordered, but this order will not be saved. To drag an image, use the handle in the top right of the image. The handle is required to allow for swipe scrolling of the slider component without moving images
